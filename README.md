> ##  master分支上是使用springboot+quartz+mysql+kafka实现定制化定时任务功能
> *   系统启动初始化加载已有的定时任务
> *   启动or暂停定时任务
> *   立即运行定时任务
> *   实时新增及修改定时任务
> *   定时任务的详细监控
> *   博客地址: [使用quartz实现定时任务定制化](https://blog.csdn.net/fanrenxiang/article/details/85539918)
> *   注意此项目需要kafka服务，新手请参照kafka官网http://kafka.apache.org/quickstart安装启动
